TECHNICAL REQUIREMENTS
═══════════════════════════════════════════════════════════════════

- Single HTML file with embedded CSS and JS (no external frameworks)
- Google Fonts: import Playfair Display + DM Sans + Space Mono
- Custom animated cursor (glowing dot + ring that follows mouse)
- IntersectionObserver for scroll-triggered animations
  (fade-in + slide-up for all sections as they enter viewport)
- CSS custom properties (variables) for all colors, used consistently
- Smooth scroll: html { scroll-behavior: smooth; }
- Mobile responsive: breakpoints at 768px and 480px
- Hamburger menu for mobile (animated toggle)
- Portfolio filter: pure JS, toggle 'active' class + CSS transition
- Progress bars animate on scroll (start at 0%, animate to target %)
- All hover transitions: 0.3s ease
- Page load: staggered hero text animation using animation-delay
